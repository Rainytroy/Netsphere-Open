# Netsphere 服务停止工具说明

## 文件说明

本目录包含两个服务停止工具：

1. `stop-dev.bat` - 原始版本，支持中文界面，但在某些系统上可能出现乱码或窗口不显示的问题
2. `stop-dev-ascii.bat` - 简化版本，只使用ASCII字符，解决乱码问题，适用于所有Windows系统

## 使用方法

### 如何使用ASCII版本停止服务

1. 右键点击 `stop-dev-ascii.bat` 
2. 选择"以管理员身份运行"
3. 在提示时按 Y 确认继续
4. 等待脚本完成所有操作

### 使用说明

- 此脚本将自动停止所有Netsphere相关服务，特别是占用端口3000、3001和3002的服务
- 脚本使用三层停止策略：常规方法、强力方法和终极方法
- 特别加强了对端口3001的处理，解决了"EADDRINUSE: address already in use :::3001"错误
- 完成后会显示操作结果

### 注意事项

- 必须以管理员身份运行才能使用所有功能，特别是网络服务重启功能
- 如果仍然无法释放端口，重启计算机可能是最有效的解决方案
- 此ASCII版本使用英文界面，避免了中文字符编码问题

## 技术信息

ASCII版本在以下几个方面进行了改进：

1. 移除了所有中文字符，避免编码问题
2. 优化了端口3001的处理逻辑
3. 增加了管理员权限检查
4. 简化了代码结构，提高了稳定性
